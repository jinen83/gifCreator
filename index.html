📖 TableGrid Component Documentation

Component: <TableGrid />

Description: A reusable table component used across apps to display structured data.

🔹 Sample API Fetch Code

fetch("https://apigenerator.dronahq.com/api/3fk8ctvP/data") .then(response => response.json()) .then(data => { const table = document.getElementById("tableBody"); table.innerHTML = ""; // Clear existing rows data.forEach(item => { const row = document.createElement("tr"); row.innerHTML = `<td>${item.id}</td><td>${item.name}</td><td>${item.email}</td>`; table.appendChild(row); }); }) .catch(error => console.error("Error fetching data:", error));

📌 Expected JSON Response:

[ { "id": 1, "name": "Alice", "email": "alice@example.com" }, { "id": 2, "name": "Bob", "email": "bob@example.com" } ]
